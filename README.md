# My Profile ReadMe
All my contributions are authored without large language models (LLMs), [without software patents](https://endsoftwarepatents.org/innovating-without-patents), [without digital restrictions management](https://www.defectivebydesign.org/drm-free), and without being privacy-invasive. And they are always released under [licenses that are free (libre) and GPL-compatible](https://www.gnu.org/licenses/license-list.en.html).

I stand with libraries, including Internet Archive, in their right to [making human knowledge available to everyone, both physically and digitally](https://www.battleforlibraries.com).
